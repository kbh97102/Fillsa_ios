# [공통] 공통 UI 요소

## 하단 광고 영역 (V2)

- 모든 화면 최하단에 텍스트 광고 영역이 존재한다.
- 제외 화면: 로그인 화면, 입력 화면 (타이핑 화면, 메모 작성 화면)
- 광고 텍스트는 중앙 정렬, "AD" 표기 필수
- 광고 영역 클릭 시 해당 광고 사이트로 이동

## 하단 내비게이션 바

- 노출 항목: Home / List / Calendar / My page
- 노출 화면: Home, List, Calendar, My page

## 상단 헤더

| 화면 | 구성 | 동작 |
|------|------|------|
| Home / List / Calendar | 로고 + 사용자 아이콘 | 사용자 아이콘 클릭 → My page 이동 / 로고 클릭 → 액션 없음 |
| My page | 로고만 | 로고 클릭 → 액션 없음 |
